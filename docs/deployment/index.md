# Deployment

Documentation is published with GitHub Pages and Jekyll at `https://zero-release.github.io/`.

The source is plain Markdown plus `_config.yml` in the main project repository. A workflow mirrors `docs/` into `zero-release/zero-release.github.io`; generated HTML is not committed to either repository.
